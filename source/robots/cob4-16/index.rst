On these pages, you can find quick start information about Care-O-bot 4 (serial no 16) in the robot house.
Our documentation covers how to start and stop the robot, charge its batteries, and how to use some of its software and remote control.
There is also a section on how to recover the robot when its emergency stop is engaged.
A comprehensive user manual and software overview can be found in the `Ros Wiki`_.
